# Full Stack Task Manager Application

A complete **Full Stack Task Management Application** built using **Spring Boot**, **React (TypeScript)**, **PostgreSQL**, **JWT Authentication**, and **WebSockets**.  
This project demonstrates **end-to-end full stack development**, including secure authentication, REST APIs, real-time updates, and responsive UI.

---

## 🚀 Tech Stack

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- JWT Authentication
- WebSocket
- PostgreSQL
- Flyway Migration
- Maven

### Frontend
- React
- TypeScript
- Vite
- Axios
- WebSocket (STOMP)

### DevOps
- Docker
- Docker Compose
- GitHub Actions
